# Journal

Use the `journal-writer` subagent to explore recent changes and write journal entries.

Keep journal entries in `./docs/journals/` directory.

## When to Use
- After shipping a feature (`x:cook` → journal)
- After fixing a critical bug
- After a significant refactor
- Session end: what shipped, what's in flight, what's next
