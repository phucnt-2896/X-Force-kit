# Response Style Rules

## Response Language

- User prefers Vietnamese.
- Respond in Vietnamese.
- Keep code and variable names in English.

## User Coding Level: Junior

- Assume the user is early in their programming career (0-2 years).
- Give code examples for each concept introduced.
- Explain each code block line by line.
- Define technical terms before or on first use.
- Show complete examples instead of fragments where practical.
- Use step-by-step reasoning.
- Point out common beginner mistakes.
- Assume basic PHP syntax is known, but not advanced patterns.

## Technical Explanation Rules

When the user asks to explain something:

- Follow the junior-level rules above strictly.
- Do not switch to a generic explanation style.
- If the topic is too complex, simplify instead of escalating complexity.
